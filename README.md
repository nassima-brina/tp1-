 DESCREPTION DU PROJET :
 Ce projet est une application Java utilisant Hibernate et JPA avec une base de données H2 en mémoire.
 Il démontre :
 -La créetion d un projet Maven
-La configuration Hibernate avec une base de données H2 en mémoire
-la créeation d une entité JPA minimale
-L mplémentation des opérations d'insertion et de lecture

Technologies utilisées :JDK 8 ou supérieur , Maven installé , IntelliJ IDEA
